# FileOptimizer

FileOptimizer - веб-приложение для оптимизации изображений и создания архивов. Проект позволяет загрузить изображение, выбрать параметры обработки и скачать оптимизированный файл. Также пользователь может загрузить несколько файлов и создать архив в формате ZIP, TAR или TAR.GZ.

## Precious source

Основной публичный репозиторий проекта используется как precious source для merge и публикации итоговой версии:

https://github.com/s02230127/python-project

Разработка ведётся в отдельных ветках, после чего изменения попадают в `main` через Pull Request.

## Решаемая задача

Проект решает задачу удобной локальной обработки файлов через веб-интерфейс. Основные возможности:

* загрузка изображений форматов JPG, PNG и WEBP;
* изменение формата выходного изображения;
* настройка качества изображения;
* изменение размера с сохранением пропорций;
* настройка яркости, контраста и резкости;
* создание архивов ZIP, TAR и TAR.GZ из нескольких файлов;
* отображение результата обработки: путь к файлу, размер до и после, процент экономии;
* переключение интерфейса между русским и английским языками.

## Используемые инструменты

Основные инструменты проекта:

* Python 3.12 — основной язык разработки;
* FastAPI — веб-сервер и обработка HTTP-запросов;
* Jinja2 — HTML-шаблоны веб-интерфейса;
* Pillow — обработка изображений;
* zipfile и tarfile — создание архивов;
* Babel — локализация интерфейса;
* Sphinx — генерация документации;
* pytest и pytest-cov — тестирование и проверка покрытия;
* Ruff — проверка стиля кода и docstring;
* doit — автоматизация тестов, сборки, документации и переводов;
* build и setuptools — сборка wheel и source distribution.

## Установка и запуск

После клонирования репозитория:

```bash
git clone https://github.com/s02230127/python-project.git
cd python-project

python3.12 -m venv .venv
source .venv/bin/activate

python -m pip install --upgrade pip
python -m pip install -e ".[dev]"
```

Запуск веб-приложения:

```bash
fileoptimizer-web
```

После запуска приложение доступно по адресу:

```text
http://127.0.0.1:8000
```

## Проверка проекта

Запуск всех основных задач:

```bash
doit
```

Отдельные команды:

```bash
doit lint
doit test
doit docs
doit build
```

После сборки дистрибутивы появляются в папке `dist/`.
